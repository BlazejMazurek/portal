Pseudoelement :first-line obejmuje formatowaniem
całą pierwszą linię.

p:first-line { color:red; font-size:20pt; font-weight:bold; }

Pseudoelement :first-letter pozwala na formatowanie pierwszej litery w bloku tekstu.

p:first-letter { color:red; font-size:100pt; }

Pseudoelementy: :after oraz :before. Poza :first-letter oraz :first-line w specyfikacji umieszczono jeszcze dwa inne pseudoelementy: :after oraz :before. Oba elementy służą do umieszczania tekstu przed i za formatowanym elementem strony.

p:before { content:"TO SĄ ĆWICZENIA"; font-size:15pt; font-style:italic; font-weight:bold; } p:after { content:"AUTOR Król Lew"; font-size:15pt; font-style:italic; font-weight:bold; }
